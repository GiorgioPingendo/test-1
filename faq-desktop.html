Pingendo Desktop - FAQ

What is Pingendo Desktop ?


Pingendo is a desktop Bootstrap builder.
It's simple and generates HTML and SASS markup compliant to Bootstrap 4.
It works on Mac, Windows and Linux

Who is it for ?


  Pingendo desktop is for developers who work with Bootstrap and want to be more productive.
It's also a great tool for learners.

How much does it costs ?

Pingendo is free.

Does an online version exists ?

Yes, Pingendo for Github and it's for workgroup
