FAQ

Pingendo

What is it ?


Pingendo is a desktop Bootstrap builder.
Simple to use, generate only Bootstrap 4 HTML and SASS compliant markup.
Works on Mac, Windows and Linux

For who is intended ?


  For developers that works with Bootstrap and wants to be more productive.
For who wants to learn Bootstrap.

How much does it costs ?

Pingendo is free.

Does an online version exists ?

Yes, It's Pingendo for Github

Can I work with files created outside Pingendo?

Although Pingendo can read any HTML file, it offer specific support only for Bootstrap 4 components and theme.

So, where to start ?

Download latest version from www.pingendo.com
