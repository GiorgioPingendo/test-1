Pingendo quick start

Pingendo is a Bootstrap HTML and SASS builder available in two versions:   

Web client - Online builder for teams and clients integrated with Github.  Lets non-devs work on Github flow. 
Desktop client - Desktop builder for developers. It works locally and offline to integrate into the developer's toolset.
