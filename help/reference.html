Pingendo web client

Pingendo is a web app that works in Chrome or Safari. Support for other browsers will be added. It integrates with Github API to load and save HTML pages and their assets in repositories. When you access Pingendo, you'll be required to grant Pingendo access to Github API. That's in order to let Pingendo read and save files stored in the repository. We use standard OAuth mechanism provided by Github and we do not store your password in any way. For additional info check out the Github website https://help.github.com/articles/connecting-with-third-party-applications/.

Page panel

You can modify SEO tag for the page and customize the Sass theme.
To check which Sass theme is attached to page or to choose a new one, look at the FILE property of the page tab. 
You can edit colors, typography and grid variables. If you need to customize more you can edit the Sass source directly.

Code editor

If you are working on something more than a quick mockup, you can get your hands dirty by using the HTML and Sass editor in the bottom tab. 
The editor is live, so the canvas will update realtime to reflect changes and vice versa.  
Pingendo works only with proper and valid HTML documents.  
Preprocessors (like PHP) or HTML fragments (like Angular templates) are not supported and won't be saved correctly.
