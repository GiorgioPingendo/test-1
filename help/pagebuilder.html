Getting started

Pingendo lets you build good-looking and perfectly coded HTML pages, deadly fast. 

To start mocking up a new page you can drag snippets of pre-made sections or stock Bootstrap 4 components.

The builder interface consists of 4 main areas: 

  • Canvas
  • Sections and components
  • Inspector
  • Code editor

Canvas

The canvas contains your document. 
You can drop elements here.
Click to select an element on canvas and play with its properties in the "SELECTION" panel.
The ESC key moves your selection one parent up.
Double click to edit text
You can rearrange the grid dragging the vertical helper.
To effectively work with the grid you should know some basics

Sections and components                              

Drag pre-made sections and Bootstrap 4 stock components.
Clicking the icons at the top of the panel switches the theme for placeholder images.
