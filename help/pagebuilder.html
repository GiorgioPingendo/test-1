Page builder

Pingendo lets you build good-looking and perfectly coded HTML pages, deadly fast.

The builder interface consists of 4 main areas: 

  • Canvas
  • Sections and components
  • Page and selection
  • Code editor

To start mocking up a new page you can drag snippets of pre-made sections or stock Bootstrap 4 components.

Canvas

The canvas show your document. 
You can dropelements ot it.
Clickto select elements and play with their properties in the "SELECTION" panel.

Double clickto edit text
You can rearrange the grid draggingthe vertical helper.
To effectively work with the grid you should have some knowledge on how it works. It's easy and you show read this.

The ESCkey moves your selection one parent up.

Sections

You can browse a curated collection of snippets
Click to insert after the selection
Drag onto canvas to insert where highlited.
The icons at the top of the panel switches the theme for placeholder images.

Components

You can browse a curated collection of snippets
Clickto insert after the selection
Dragonto canvas to insert where highlited.
The icons at the top of the panel switches the theme for placeholder images.

Page panel

You can modify SEO tag for the page and customize the Sass theme.
To check which Sass theme is attached to page or to choose a new one, look at the FILE property of the page tab. 
You can edit colors, typography and grid variables. If you need to customize more you can edit the Sass source directly.  

Selection panel

Show information about currently selected element.
The top part show the hierarchy 
The properties include all the Bootstrap 4 properties.


Code editor

If you are working on something more than a quick mockup, you can get your hands dirty by using the HTML and Sass editor in the bottom tab. 
The editor is live, so the canvas will update realtime to reflect changes and vice versa.  
Pingendo works only with proper and valid HTML documents.  
Preprocessors (like PHP) or HTML fragments (like Angular templates) are not supported and won't be saved correctly.
