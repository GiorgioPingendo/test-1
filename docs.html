Introduction

Pingendo is a Bootstrap builder available in two version, one to work locally on filesystem and the other one that works online integrated with Github. 


Pingendo Desktop

Pingendo Desktop it's an app that works on Windows (9+), OSX (10.9+) and Linux.  
You can get a copy from www.pingendo.com/desktop.html  

Once installed and launched you can choose to create a page from a template or open an existing HTML file.
For learn more about what you can do in Pingendo jump to the Editor section


Pingendo for Github

Pingendo for Github it's a web app that works in Chrome or Safari. Support for other browsers will be added.
