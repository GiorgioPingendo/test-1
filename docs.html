Introduction

Pingendo is a Bootstrap builder available in two version, one to work locally on filesystem and the other one that works online integrated with Github. 


Pingendo Desktop

Pingendo Desktop it's an app that works on Windows (9+), OSX (10.9+) and Linux.  
You can get a copy from www.pingendo.com/desktop.html  

Once installed and launched you can create a page from a template. 
The result is saved as plain HTML, Sass and CSS files that you can modify also with external editors. No need to export or import nothing.
To learn more about what you can do in Pingendo jump to the Editorsection


Pingendo for Github

Pingendo for Github it's a web app that works in Chrome or Safari. Support for other browsers will be added.
Integrates with Github API to load and save HTML pages and theirs assets in repositories.
First time you'll try to log in Pingendo you'll be prompted to grant it access to Github API. This is required in order to let PIngendo read and save files.
We use standard OAuth mechanism provided by Github and we do not store in any way your password.
For additional info check out the Github website https://help.github.com/articles/connecting-with-third-party-applications/

Builder

The builder interface it's the same for both PIngendo Desktop and Pingendo for Github and consists of 5 main areas 

1 the toolbar 

2 sections and components panel permit you to browse and drag premade sections and Bootstrap 4 components

3 the canvas where you can drop elements 

4 the property panel that permit to change setting for the selection and the page

5 the source panel let you work on 




HTML

Pingendo let you build good-looking and perfectly coded HTML pages, deadly fast. 

To start mocking up a new page you can drag needed elements from sections and components panel.
Sections are pre-made sections commonly used in site development. 
Components are all the ones that Bootstrap 4 features 

You can click on canvas to select single elements and play with its properties using the selection tab of the property panel.   The properties are the ones that Bootstrap offers. (So don't blame us if something is counterintuitive)  

When an element is selected you can select its parents using the breadcrumb.
This is particularly useful when you can't pick an element directly from canvas.
You can also use the ESC key.

If you are working on something more than a quick mockup, you can get your hand dirty using the HTML editor in the bottom HTML tab.

The editor is live, so the canvas will update almost realtime to reflect changes and vice versa. 
Since what you are editing it's a live HTML DOM, preprocessors (like PHP) or custom templates syntaxs are not supported.  You should  use Pingendo only on proper HTML static documents.
