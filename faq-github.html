Pingendo for Github - FAQ

What is it ?


Pingendo for Github is an online HTML builder based on Bootstrap and Github. 
It's a web app that simplify github.com usage, in order to build static websites.

For who is intended ?


  For developers and agencies who wants to share with  their clients 
For marketing guys who needs to create prototypes of landing pages or small sites.

How does it works ?

Once logged, users will be able to browse HTML pages in Github repos, make their changes, submitting them as pull requests without even knowing what Github or Bootstrap are. 

Everyone can contributes according to its own role and skill. A common use case might be: Basic users update text and images, advanced users quickly create entire webpages and developers add custom behaviour to the vanilla Bootstrap source. 

Can i use with private repositories ?

Not yet, we are working on. 

Can i use it from iOS or Android devices ?

Not yet, we are working on. 

How much does it costs ?

Pingendo for Github is free during beta period. Pricing will be per repo.

So, how to start ?

Just point to  v4.pingendo.com/github.html
