What is it ?


Pingendo for Github is an online Bootstrap builder that work with Github.

For who is intended ?


  For developers and agencies that wants to let their users review and modify real HTML pages hosted on Github instead of submitting PSD and colleting feedback via mail.

How does it work ?

Pingendo for Github is a web app. Once logged, users will be able to browse HTML pages in github repos, make their changes and submitting them as pull requests without even knowking what  Github and Bootstrap are
