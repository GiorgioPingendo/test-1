Pingendo for Github - FAQ

What is Pingendo for Github ?


It's an online WYSIWYG Bootstrap builder integrated with Github.

Who is it for ?

Developers and agencies who want their team and clients to review and contribute in static sites development.

How does it work ?


Everyone can browse and edit HTML pages hosted in Github repositories with a simple interface.
Changes made with the WYSIWYG editor will be submitted as pull requests.
No need to know what Github or Bootstrap are.

Is it ok for newbies too? 


It's easy enough for everyone to perform basic tasks as change text or images.
Making webpages from scratch is not that far because Pingendo is a great tool to learn HTML/CSS.

Which are the benefits ?

Developers will manage users contributions as pull-requests.
Marketing and communication will enjoy an unseen independence. 
Web agencies will benefit of a smoother workgroup iteration.
Stakeholders will better understand live webpages than static mockups.

Can i use it like a CMS ?


Yes, if you are building a static web site. We also integrated Github Pages for free hosting. 

Does it works on private repositories ?

Not yet, we are working on it. 

Does it works on iOS or Android ?

Not yet, we are working on it. 

How much does it costs ?

Pingendo for Github is free while in beta.

So, where to start ?

Just point your browser to  v4.pingendo.com/github.html
