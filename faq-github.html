Pingendo for Github - FAQ

What is the problem Pingendo solves ?


Back and forth of psd, screenshot, mockups, and every sort of offline docs are today the backbone of teams iteration during the creation of wepages.
Stakeholders approvals and contribution by copywriters, designers, customers or other resources, mostly happen offline.
Developer is than required to take everything online, even easy modification or unclear requests.
Wait a moment, developers are not the scarcer resource?
Why do not involve developers only with the 10% code and leave the 90% content iteration to the marketing and communication team?

What is the solution?


Everyone working on the same HTML/CSS source that from early prototype will evolve into production.
Developers works on code iterations only. Marketing and communication works on content, UI and UX. 
Everything managed by the revision platform Github. 

                         

How the solution works?

A boostrap 4 builder is the tool that enables copywriters to add text or modify sizes, and enables advanced users to quickly create webpages. Everyone will act according to its own role and skill.

A revision UI hides the complexity of Github. Users propose the revision to the project owner that will accept or deny it.
Messed up? Developers has all the power of Github to fix it.

                         

What is it ?


Pingendo for Github is an online Bootstrap builder integrated with Github.
                         

Who is it for ?


  For developers and web agencies who want to let their users review and modify static HTML pages hosted on Github.
An alternative to back and forth of PSD, emails, docs and so on.

How does it work ?

Pingendo for Github is a web app that simplify github.com usage. Once logged, users will be able to browse HTML pages in Github repos, make their changes, submitting them as pull requests without even knowing what Github or Bootstrap are.

Can i use on private repositories ?

Not yet, we are working on. 

How much does it costs ?

Pingendo for Github is free during beta period. Pricing will be per connected Github repository.

So, how to start ?

Just point to  v4.pingendo.com/github.html
