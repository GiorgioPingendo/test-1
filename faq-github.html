Pingendo for Github - FAQ

What is it ?


Pingendo for Github is an online Bootstrap builder that integrates with Github.

For who is intended ?


  For developers and agencies that wants to let their users review and modify static HTML pages hosted on Github instead of submitting PSD and colleting feedback via mail or similar.

Why you need to authorize  ?

Pingendo for Github is a web app that simplify github.com usage. Once logged, users will be able to browse HTML pages in Github repos, make their changes and submitting them as pull requests without even knowking what Github and Bootstrap are.

Does it works on private repositories ?

Not yet, we are working on. 

How much does it costs ?

Pingendo for Github is free during beta period. Pricing will be per repo.
