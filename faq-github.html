Pingendo for Github - FAQ

What is Pingendo for Github ?


It's a dead simple web-based Bootstrap builder integrated with Github.

Who is it for ?

Developers and agencies who would like to let their team and clients to review and contribute on HTML pages since the first draft.

How does it work ?


Everyone can browse and edit HTML pages hosted in Github repositories with a simple interface.
Changes made with the WYSIWYG editor will be submitted as pull requests.
No need to know what Github or Bootstrap are.

How does it works ?

Pingendo for Github integrates in one simple interface what is needed to build and publish static web sites with Bootstrap and Github.
Everyone can browse pages, propose his changes as pull request and publish updates to Github Pages.  
Editing of HTML with Pingendo is easy, you can test it here.

Can i use it like a CMS ?


Yes, if you work on static web sites made with Bootstrap and hosted on Github Pages.

Does it works with private repositories ?

Not yet, we are working on. 

Does it works on iOS or Android ?

Not yet, we are working on. 

How much does it costs ?

Pingendo for Github is free during beta period. Pricing will be per repo.

So, how to start ?

Just point to  v4.pingendo.com/github.html
