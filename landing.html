One app, two flavours

Pingendo  targets both developers and non-developers, that needs to work together on HTML prototypes.
For this reason the same app is packed in 2 version: one as desktop app for developers that needs to use it offline and one for the others that works directly on Github repo from a browser

Works offline on plain HTML and SASS files.
Easily integrates in developer toolset

Download

Pingendo for Web enables everyone to review and edit HTML pages on Github repositories, helping teams of developers and non-developers to iterate faster.

Organize proposals of change in Pull Requests, so that developers can tidily and safely merge changes.
