Instant setup

All you need is a valid Github account. No installations required, it works on any Github repository.

Git Backed

Every project is hosted in a Github repository. Developers manage users within Github and checkout the project locally.

Safe collaboration

Changes  are submitted as pull-requests and reversible at any time. Developers can spend time building, not worrying.

Publish For Free

Publish anywhere or with a single click on Github Pages, the free hosting service kindly offered by Github.

Project leader 

A non-dev might be set as project leader to merge/discard non-devs contributions. It will save even more dev time.
