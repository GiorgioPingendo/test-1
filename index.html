Hyper-simple Bootstrap website builder for developers and their clients.



Pingendo is a collaborative drag-and-drop Bootstrap builder.

Its simplicity enables both developers and non-developers to build static websites 
together, each one contributing according to its own role and skill.

Efficient collaboration on Github

Pingendo is easy enough to take everyone on Github since first webpage scratch. 
Teammates as clients, stakeholder or content makers view pages and submit edits as pull-request on Github. 
In short: non-techie take care of the time-consuming minor changes while devs work on source on Github as usual.
