Pingendo for Github is a web app suitable for basic changes of text and images as well for webpage developing.
It's integrated with Github to work on repositories and it saves changes from users as pull requests.
It enables clients, stakeholders, marketing and communication people to contribute easily and safely on the creation of static websites.

Visual editor

The editor works on static HTML and SASS files, allowing simple modification of contents for non developers as well as full authoring for developers.

Integrated with Github

Make dead-simple to navigate repositories and propose changes as pull requests.

Efficent workflow

Take everyone on Github for efficent collaboration of mixed teams. 
Non-developers can work indipendently while developers works clen pull request

Safe

Developers manage users permissions and contributions on Github. Revisions are revertable at any time.
