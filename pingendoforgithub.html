PINGENDO FOR GITHUB

Pingendo for Github enables everyone from a browser to review and modify Bootstrap prototypes hosted on Github. 


GETTING STARTED IT'S EASY

Point your browser to the example repository that we set up for you
github.pingendo.com/Pingendo/example-singlepage

Authorize Pingendo to access Github repositories and fork the example repo.

You are ready to push your prototypes !

So i.e. github.com/twbs/bootstrap will be available at github.pingendo.com/twbs/bootstrap
