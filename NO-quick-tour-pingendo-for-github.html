Pingendo for Github

Pingendo is a collaborative Bootstrap builder. Enables non-developers to contribute-developers since first prototype.

Unlike common CMSs, Pingendo works on both layout and content.

Bring users on Github


When working on static pages or prototypes, Pingendo is easy enough to permit most users to browse github repostiories, modify pages and submit changes as pull-request. So developer and non developers can profitably collaborate speaking the same language.
